# 17분 후의 지구 — 프로토타입

모바일 세로 화면을 기준으로 만든 타임루프 재난 서사 게임 프로토타입입니다. 17분마다 반복되는 도시 붕괴를 배경으로, 세 인물이 예지·기억·신뢰·집중력·확보 물자를 활용해 균열을 막는 구조를 실험합니다.

> **지층이 갈라지고 불이 꺼졌다.** 경보도 없이, 도심 한가운데서. 연기 속 어딘가에서 세 사람에게 무언가가 닿았다 — 그 순간부터, 세 사람은 아직 일어나지 않은 것들을 볼 수 있게 됐다.

## 핵심 루프

1. 17분 타이머가 흐르며 재난 단계가 진행됩니다.
2. 예지를 사용해 미래를 관측합니다.
3. 중간 장면에서 세 인물 중 누구의 방식으로 개입할지 선택합니다.
4. 루프 종료 시 일부 자원과 기억이 계승됩니다.
5. 기억 조각이 쌓이면 진원의 좌표와 엔딩 조건이 열립니다.

## 등장인물

| 인물 | 역할 | 루프에서 하는 일 |
|---|---|---|
| 서가람 | 예지 관측자 | 다음 붕괴를 먼저 읽고 위험을 경고합니다. |
| 윤도현 | 실행자 | 중간 장면에서 직접 개입해 경로를 열고 균열을 꺾습니다. |
| 이세아 | 계승자 | 루프마다 기억 조각을 쌓아 진원의 좌표를 잇습니다. |

## 현재 구현된 기능

- 모바일 9:16 카드형 UI
- 17분 루프 타이머와 속도 조절(1×/10×/60×)
- 균열 게이지와 패배 조건
- 3인 캐릭터 패널
- 5자원 HUD: 예지, 기억 조각, 신뢰, 집중력, 확보 물자
- 3단계 타임라인
- 미래 관측 패널과 관측 로그
- 중간 장면 분기 선택 및 숙련 누적
- 루프 계승 규칙
- 진원 해금, 붕괴, 엔딩 도감

## 프로젝트 구조

```text
.
├── index.html      # 모바일 UI 구조
├── styles.css      # 다크 테마/모바일 스타일
└── src/
    ├── data.js     # 캐릭터, 자원, 구간, 엔딩, 튜토리얼 데이터
    └── game.js     # 루프 진행, 상태 관리, UI 인터랙션
```

## 실행 방법

```bash
git clone https://github.com/perust/17min-earth-proto.git
cd 17min-earth-proto
python -m http.server 8000
```

브라우저에서 `http://localhost:8000`에 접속합니다. 모바일 화면 비율로 보는 것을 권장합니다.

## 개발 메모

- 별도 빌드 도구 없이 ES module 기반으로 동작합니다.
- 데이터와 밸런스는 `src/data.js`, 실제 진행 로직은 `src/game.js`에 분리되어 있습니다.
- 프로토타입 단계이므로 수치와 엔딩 조건은 빠르게 조정될 수 있습니다.
