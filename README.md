# 17분 후의 지구 — 프로토타입

**세계가 무너지기 17분 전.** 지층이 갈라지고 불이 꺼진다. 17분 뒤 붕괴가 완성되면 시간은 처음으로 되감긴다 — 그 고리 안에 세 사람이 있다.

| 인물 | 역할 | 루프에서 하는 일 |
|------|------|-----------------|
| **서가람** | 관측자 | 예지를 소모해 다음 붕괴를 먼저 읽는다 |
| **윤도현** | 실행자 | B구간에서 한 번 개입해 경로를 열고 균열을 꺾는다 |
| **이세아** | 계승자 | 기억 조각을 루프마다 쌓아 진원의 좌표를 잇는다 |

루프마다 기억 조각이 쌓이면 진원의 좌표가 드러나고, 고리를 끊을 문이 열린다. 균열 게이지가 100에 닿기 전에.

## 포함 파일
- `index.html` — 화면 구조
- `styles.css` — 모바일 우선 스타일
- `src/data.js` — 3인 캐릭터 / 5자원 / 타임루프 상수 / 튜토리얼
- `src/game.js` — 루프 UI와 기본 인터랙션

## 실행 방법
간단한 로컬 서버로 열면 됩니다.

```bash
cd /opt/data/17분후의지구_proto
python3 -m http.server 8000
```

그 다음 브라우저에서:

```text
http://127.0.0.1:8000
```
